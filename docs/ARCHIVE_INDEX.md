## Tombstone Rules (from ID_REGISTRY.md)

When an entity is removed from the main ID registry:
1. Its ID becomes a tombstone (marked 🗑️)
2. The removal is logged in the decision log (D-13)
3. The tombstone entry is added to ID_REGISTRY.md's Tombstones section
4. IDs are never reused
